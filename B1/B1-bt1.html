-HTML BASIC
<!DOCTYPE>:
Xác định loại tài liệu và giúp trình duyệt hiển thị trang web chính xác. Nó phải xuất hiện một lần duy nhất ở đầu trang, không phân biệt chữ hoa chữ thường.
HTML Headings:
HTML sử dụng thẻ <h1> đến <h6> để xác định tiêu đề.
<h1>: là tiêu đề quan trọng nhất
<h6>: là tiêu đề ít quan trọng nhất
HTML Paragraphs:
HTML sử dụng thẻ <p> để xác định đoạn văn bản.
HTML Links:
HTML sử dụng thẻ <a> để xác định liên kết.
HTML Images:
HTML sử dụng thẻ <img> để hiển thị hình ảnh, với các thuộc tính quan trọng như src, alt, width và height.
-HTML Elements
HTML Elements:
Một phần tử HTML bao gồm thẻ mở, nội dung và thẻ đóng.
<h1> Tiêu đề: <h1>My First Heading</h1>
<p> Đoạn văn: <p>My first paragraph.</p>
Nested HTML Elements:
Các phần tử HTML có thể lồng nhau, và mọi tài liệu HTML đều chứa các phần tử này. Ví dụ phổ biến gồm <html>, <body>, <h1> và <p>.
Never Skip the End Tag:
Một số phần tử HTML vẫn hiển thị đúng ngay cả khi bạn quên thẻ đóng, nhưng luôn nên viết đầy đủ để đảm bảo trang web hoạt động chính xác.
Empty HTML Elements:
Các phần tử HTML không có nội dung được gọi là phần tử rỗng.
-HTML Attributes
The href Attribute
Thẻ <a> dùng để tạo liên kết (hyperlink).Thuộc tính href xác định URL của trang đích.
The src Attribute
Thẻ <img> dùng để nhúng hình ảnh vào trang HTML. Thuộc tính src xác định đường dẫn đến hình ảnh.
Có hai cách để xác định URL trong thuộc tính src của thẻ <img>
URL tuyệt đối: Dẫn đến ảnh từ trang web khác. Ví dụ: src="https://www.w3schools.com/images/img_girl.jpg" Lưu ý: Ảnh bên ngoài có thể bị xóa, thay đổi hoặc vi phạm bản quyền.
URL tương đối: Dẫn đến ảnh nằm trong trang web của bạn. src="img_girl.jpg" → Ảnh trong thư mục hiện tại. src="/images/img_girl.jpg" → Ảnh trong thư mục /images/, tính từ gốc trang web.
The width and height Attributes
Thẻ <img> nên có các thuộc tính width và height để xác định kích thước hình ảnh (tính bằng pixel).
The alt Attribute
Thuộc tính alt trong thẻ <img> cung cấp văn bản thay thế khi hình ảnh không hiển thị do lỗi kết nối, sai đường dẫn hoặc dành cho người dùng sử dụng trình đọc màn hình.
The style Attribute
Thuộc tính **`style`** được dùng để thêm **CSS trực tiếp** vào phần tử, như thay đổi **màu sắc, phông chữ, kích thước**, v.v.
The lang Attribute
Bạn nên luôn thêm thuộc tính lang trong thẻ <html> để khai báo ngôn ngữ của trang web. Điều này giúp công cụ tìm kiếm và trình duyệt xử lý nội dung tốt hơn.
The title Attribute
Thuộc tính title cung cấp thông tin bổ sung về một phần tử.Giá trị của title sẽ hiển thị dưới dạng chú giải (tooltip) khi di chuột qua phần tử.
-HTML Headings
HTML Headings
<h1>: là tiêu đề quan trọng nhất
<h6>: là tiêu đề ít quan trọng nhất
Headings Are Important
Tiêu đề giúp công cụ tìm kiếm lập chỉ mục và giúp người dùng đọc lướt dễ dàng. Sử dụng <h1> cho tiêu đề chính, <h2> cho tiêu đề phụ, và các mức thấp hơn cho nội dung ít quan trọng hơn.
Bigger Headings
Mỗi tiêu đề HTML có kích thước mặc định, nhưng bạn có thể tùy chỉnh kích thước bằng thuộc tính style với CSS font-size.
-HTML Headings
HTML Paragraphs
Thẻ <p> dùng để xác định đoạn văn trong HTML. Mỗi đoạn văn bắt đầu trên dòng mới, và trình duyệt tự động thêm khoảng trắng (margin) trước và sau đoạn văn.
HTML Display
Cách hiển thị HTML không cố định, nó thay đổi tùy theo kích thước màn hình và cửa sổ trình duyệt. Thêm khoảng trắng hoặc dòng trống trong mã HTML sẽ không ảnh hưởng đến hiển thị. Trình duyệt tự động xóa khoảng trắng và dòng trống thừa khi hiển thị trang.
HTML Horizontal Rules
Thẻ <hr> tạo đường kẻ ngang để phân tách nội dung và không có thẻ đóng.
HTML Line Breaks
Thẻ <br> tạo dấu ngắt dòng, giúp xuống dòng mà không cần tạo đoạn văn mới. Là thẻ rỗng, không có thẻ đóng.
Solution - The HTML <pre> Element
Thẻ <pre> hiển thị văn bản được định dạng sẵn, giữ nguyên khoảng trắng và dấu xuống dòng.Nội dung bên trong sử dụng phông chữ có chiều rộng cố định (thường là Courier).
- HTML Paragraphs
HTML Paragraphs
Phần tử <p > trong HTML xác định một đoạn văn. Đoạn văn luôn bắt đầu trên một dòng mới và trình duyệt tự động thêm khoảng trắng (margin) trước và sau nó.
HTML Display
Bạn không thể chắc chắn cách HTML sẽ hiển thị. Màn hình lớn, nhỏ hoặc cửa sổ thay đổi kích thước sẽ tạo ra kết quả khác nhau. Trong HTML, bạn không thể thay đổi hiển thị bằng cách thêm khoảng trắng hoặc dòng trống trong mã. Trình duyệt sẽ tự động loại bỏ các khoảng trắng và dòng trống thừa khi hiển thị trang.
HTML Horizontal Rules
Thẻ <hr> xác định một ngắt dòng theo chủ đề trong trang HTML, và thường được hiển thị dưới dạng một đường kẻ ngang. Phần tử <hr> được sử dụng để phân tách nội dung (hoặc xác định một sự thay đổi) trong trang HTML.
HTML Line Breaks
Phần tử <br> trong HTML dùng để ngắt dòng mà không bắt đầu một đoạn văn mới.

- HTML Text Formatting
HTML <b> and <strong> Elements

<b> thường dùng để in đậm văn bản mà không mang ý nghĩa về mặt ngữ nghĩa.

<strong> không chỉ làm văn bản in đậm mà còn biểu thị rằng văn bản đó quan trọng hơn về mặt ngữ nghĩa.

HTML <i> and <em> Elements

<i> thường dùng để in nghiêng văn bản mà không mang ý nghĩa ngữ nghĩa.

<em> làm văn bản in nghiêng và cũng thể hiện rằng văn bản đó có ý nghĩa nhấn mạnh (emphasis).

HTML <small> Element
Phần tử <small> trong HTML xác định văn bản có kích thước nhỏ hơn.
HTML <mark> Element
Phần tử <mark> trong HTML xác định văn bản cần được đánh dấu hoặc làm nổi bật.
HTML <del> Element
Phần tử <del> trong HTML xác định văn bản đã bị xóa khỏi tài liệu. Các trình duyệt thường hiển thị văn bản bị xóa bằng cách gạch ngang qua nó.
HTML <ins> Element
Phần tử <ins> trong HTML xác định văn bản đã được chèn vào tài liệu. Các trình duyệt thường hiển thị văn bản được chèn bằng cách gạch chân.
HTML <sub> Element
Phần tử <sub> trong HTML dùng để hiển thị văn bản chỉ số dưới, thường dùng cho công thức hóa học như H2O.
HTML <sup> Element
Phần tử <sup> trong HTML dùng để hiển thị văn bản chỉ số trên, thường dùng cho chú thích chân trang như WWW[1].
- HTML Quotation and Citation Elements
HTML <blockquote> for Quotations
Phần tử <blockquote> trong HTML dùng để trích dẫn từ nguồn khác và thường được thụt lề trong trình duyệt.
HTML <q> for Short Quotations
Thẻ <q> trong HTML xác định một đoạn trích ngắn. Các trình duyệt thường thêm dấu ngoặc kép xung quanh đoạn trích.
HTML <abbr> for Abbreviations
Thẻ <abbr> trong HTML dùng để đánh dấu từ viết tắt hoặc tên gọi tắt, cung cấp thông tin cho trình duyệt và công cụ tìm kiếm
HTML <address> for Contact Information
Thẻ <address> trong HTML dùng để xác định thông tin liên lạc của tác giả hoặc chủ sở hữu tài liệu, thường được hiển thị in nghiêng với dòng ngắt trước và sau.
HTML <cite> for Work Title
Thẻ <cite> trong HTML dùng để xác định tiêu đề của một tác phẩm sáng tạo, thường hiển thị in nghiêng
HTML <dob> for Bi-Directional Overrid
DO là viết tắt của Bi-Directional Override. Thẻ <dob> trong HTML được sử dụng để ghi đè hướng văn bản hiện tại.

- HTML Styles - CSS
What is CSS?
CSS được sử dụng để định dạng bố cục trang web, bao gồm màu sắc, phông chữ, kích thước văn bản, khoảng cách, hình nền, và hiển thị cho các thiết bị và kích thước màn hình khác nhau
Using CSS
CSS có thể được thêm vào HTML theo 3 cách: inline (thuộc tính style trong phần tử), internal (thẻ <style> trong phần <head>), và external (thẻ <link> liên kết với file CSS ngoài). Cách phổ biến là sử dụng file CSS ngoài.
Inline CSS

CSS nội tuyến áp dụng kiểu dáng cho một phần tử HTML duy nhất thông qua thuộc tính style. VD:

<h1 style="color:red;">This is a heading</h1>

This is a heading

Internal CSS

CSS nội bộ được sử dụng để định nghĩa kiểu dáng cho một trang HTML duy nhất.

CSS nội bộ được định nghĩa trong phần <head> của trang HTML, trong thẻ <style>.

Ví dụ dưới đây đặt màu văn bản của tất cả các phần tử <h1> (trên trang đó) thành màu xanh dương, màu văn bản của tất cả các phần tử <p> thành màu đỏ, và trang sẽ có nền màu "powderblue".

External CSS

Bảng kiểu ngoại được sử dụng để định nghĩa kiểu dáng cho nhiều trang HTML.

Để sử dụng bảng kiểu ngoại, bạn cần thêm một liên kết đến nó trong phần <head> của mỗi trang HTML.
