HTML Tutorial

HTML là ngôn ngữ đánh dấu chuẩn để tạo trang web. Học HTML giúp bạn hiểu cấu trúc cơ bản của web.

Giới thiệu về HTML

HTML viết tắt của HyperText Markup Language, là ngôn ngữ dùng để xây dựng trang web.

Cấu trúc cơ bản

Một trang HTML thường có cấu trúc như sau:

<!DOCTYPE html>
<html>
<head>
    <title>Tiêu đề trang</title>
</head>
<body>
    <h1>Chào mừng đến với HTML</h1>
</body>
</html>

Bạn đã cuộn được một đoạn dài!
